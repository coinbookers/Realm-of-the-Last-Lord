# Realm-of-the-Last-Lord
Game called Realm of the Last Lord. It is designed as a lightweight fantasy simulation where players make decisions that affect survival and progression inside a collapsing kingdom. The project is intentionally built using only the C++ standard library to ensure portability and clarity for beginners.

The story is set in a fractured world where a forgotten lord tries to restore order while resisting pressure from ancient powers that once shaped civilization. Meanwhile, scattered groups of humans attempt to survive in unstable regions of the realm. The gameplay focuses on decision-making rather than graphics or complex mechanics.

The code demonstrates structured programming principles such as function separation, state tracking, random event generation, and simple resource systems. Each run produces a different outcome, encouraging replayability and experimentation.

The goal of this repository is educational: to show how a complete interactive program can be built in C++ without external dependencies. Developers are encouraged to modify the logic, expand the world system, or add additional events.
